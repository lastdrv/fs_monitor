Мониторинг файловой системы на предмет "кто сожрал место"
это на случай если df и ncdu не помогает.

Формат выходного файла:
1. дельта между "было" и "стало"
2. текущий размер файла, если файла раньше не было, то ставится прочерк
3. файл который пожирнел

В выходной файл пишутся только те файлы, размер которых увеличился.
